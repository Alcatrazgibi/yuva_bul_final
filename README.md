   Kullanılan Veri Tabanları  : Firebase Veri Tabanı
   
   Kullanılan Yazılım Dilleri : React Native

   Backend'de Kullanılanlar   : Node.js tabanlı React Native ekosistemi

   Projenin Temel Özellikleri :	Hayvan Sahiplendirme Amaçlı ve Kullanıcı Dostu Bir Proje 

   Test Edilen Platformlar    : Android Emulator ve Fiziksel Cihaz (Android)
